Använd dina recept för att skapa en matsedel

Om du vill skapa en matsedel automatiskt så använd nedanstående sökkriterier för att ange utifrån vilka kriterier matsedeln skall slumpas.

  1. Ange hur många recept som skall ingå i din matsedel (1-10 dagar)
  2. Ange taggar, som "vardag" eller "fisk" om ska eller inte ska ingå.
